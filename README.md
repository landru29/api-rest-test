# API REST C++


## Prerequisite

You must have a sane installation of 
* GCC
* cmake

You must install following libs:
* boost
* nlohmann/json


## Build

```bash
cmake -B build .
cmake --build build
```

## Run

```bash
./build/RestfulApi
```

## Developping

The project is configured to be used with VSCode and devcontainers (docker must be installed first).
